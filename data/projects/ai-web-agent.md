# AI Web Agent for Intelligent Search & Data Extraction

An autonomous web agent for multi-source search and data synthesis. The agent uses LLM-based
decision workflows to plan and direct its own search strategy across multiple sources, combined
with Scrapy-powered data pipelines to extract and structure the results — producing cited,
structured responses rather than raw scraped text.

Stack: Python, LLM APIs, FastAPI, Scrapy, Next.js.
