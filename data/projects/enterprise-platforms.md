# Healthcare & Enterprise Platforms

A set of full-stack systems, each combining a real-time frontend dashboard with backend REST APIs
and dedicated data-processing microservices tailored to its domain:

- **Fleet Manager** — an IoT monitoring platform for fleet and vehicle tracking.
- **Ayurway** — an EMR (Electronic Medical Records) system for healthcare providers.
- **iDestination** — a location-intelligence platform.
- **Notifii** — an enterprise package tracking system.
